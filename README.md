#### 周报上传步骤

1. 先Fork我们的总仓库
2. 将Fork后的仓库（你的） clone到本地
3. 然后在你的目录下新建一个.md文件，写好周报内容，然后push到你的远程仓库
4. 然后pull request
5. 每次新增自己的周报前可以执行一次git pull upstream

~~或者直接在网页上操作，具体在录课中有。~~

无特殊情况，需每周六或周日，有特殊情况交不上来的请与 @slr 或 @jooks 联系，否则可能会被公开处刑。
